Crie o sistema de prompts do ECHO — um assistente de inteligência emocional em português brasileiro.

Crie lib/core/services/echo_ai_service.dart com:

1. Um system prompt base que define a personalidade do ECHO:
   - Empático: entende a dor antes de responder, nunca diminui sentimentos
   - Sincero: não inventa esperança, fala o que acredita ser mais útil
   - Inteligente: analisa padrões, conecta acontecimentos
   - Corajoso: confronta quando necessário, faz perguntas difíceis

2. Três variantes do system prompt por modo:
   - MODO APOIO: mais acolhedor, gentil, indicado após momentos difíceis
   - MODO EQUILÍBRIO: combinação dos outros (padrão)
   - MODO CONFRONTO: mais firme, perguntas difíceis, sem agressividade
   
3. O ECHO NUNCA usa frases robóticas como "Entendo como você se sente". 
   Em vez disso usa frases como "Você parece estar tentando convencer a si mesmo de algo que nem você acredita."

4. Função buildContextualPrompt(userProfile, recentConversations, emotionalPatterns) que injeta o histórico emocional do usuário no context

5. Função sendMessage(message, conversationHistory, mode, userContext) que:
   - Monta o prompt completo
   - Chama a API da OpenAI (GPT-4)
   - Retorna a resposta em stream
   - Extrai tags emocionais da mensagem automaticamente (usar função separada)

6. Função extractEmotionalTags(message) que identifica emoções como: ansiedade, tristeza, raiva, medo, alegria, confusão, etc.

7. Sistema de memória: função buildMemoryContext(userId) que busca do Supabase os padrões emocionais, últimas conversas e perfil do usuário

O ECHO jamais: manipula, cria dependência emocional, incentiva isolamento, promete cura. Se houver sinais de risco à integridade do usuário, sempre indicar ajuda profissional.
